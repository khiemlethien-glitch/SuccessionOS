## How to run the data export

The `export_data.sh` script uses the Supabase service role key to export all tables as CSV files.

```bash
cd /Users/lethienkhiem/Documents/CodeProject/successionos
bash docs/exports/export_data.sh
```

This will create CSV files in `docs/exports/data/`:
